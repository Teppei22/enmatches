## OVERVIEW
業界大手のクラウドソーシングサイトは選択肢が多く複雑で操作がしにくい.そこでこれを解消する「主婦でも使えるメルカリのようでシンプルなエンジニア案件マッチングサイト」を企画し90日間で開発しました．

## 実装機能一覧
- ログイン・ログアウト機能
    - 自動ログイン機能
    - パスワードリマインダー機能
- ユーザ登録・編集・退会
- 案件登録・編集・応募・削除・いいね機能・SNSシェア・一覧表示
    - お知らせメール送信機能（応募時）
    - チャット・メッセージ送信機能（非同期処理[ajax]）
- 画像アップロード
